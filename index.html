How Quick Sort Works:

  1. Choose a pivot element (last element)
  2. Partition array: elements < pivot go left,> pivot go right
  3. Recursively sort left and right partitions
  4. Combine results (array becomes sorted)

Industry Usage: Used in Java's Arrays.sort() for primitive types
